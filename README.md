# Site Débora Kulczar Advocacia

## Estrutura
- Imagem do escritório à esquerda e textos à direita.
- Seção Áreas de Atuação com blocos responsivos.
- Bloco Advogada Responsável com foto e botão.

## Como usar
1. Adicione as imagens: `escritorio-moderno.jpg` e `perfil.jpg` na pasta do projeto.
2. Substitua os arquivos no GitHub.
3. Publique no GitHub Pages.

## Personalização
- **Cores**: ajuste no `style.css`.
- **Textos**: edite no `index.html`.
- **Botão**: ação personalizada no `script.js`.
