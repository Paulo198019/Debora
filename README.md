# Landing Page Jurídica

Este projeto é uma landing page moderna para escritórios de advocacia.

## Recursos
- Banner full screen com overlay azul.
- Logo no canto esquerdo.
- Botões estilizados em azul.
- Layout responsivo.

## Estrutura
- index.html
- style.css
- script.js
- README.md

## Como usar
1. Substitua `banner.jpg` pela imagem do seu escritório.
2. Substitua `logo.png` pela sua logo.
3. Abra `index.html` no navegador ou use um servidor local.

## Personalização
- Paleta de cores: ajuste no `style.css`.
- Textos: edite no `index.html`.
