# Débora Kulczar Advocacia - Site Institucional

Este projeto é um site institucional desenvolvido para apresentar informações sobre o escritório **Débora Kulczar Advocacia**, suas áreas de atuação, equipe e formas de contato. O design é responsivo, moderno e inclui animações suaves para melhor experiência do usuário.

---

## ✅ Funcionalidades
- **Barra Superior**:
  - Logo ajustado.
  - Telefone destacado.
  - Botão para contato por **Email**.
  - Botão para contato via **WhatsApp**.

- **Banner Full Screen**:
  - Imagem de fundo cobrindo toda a largura e altura da tela.
  - Texto sobreposto com mensagem institucional.

- **Seções**:
  - **Quem Somos**: Imagem à esquerda e texto à direita.
  - **Áreas de Atuação**: Cards organizados em duas linhas (3 blocos na primeira, 2 na segunda) com ícones e descrições.
  - **Advogada Responsável**: Título, subtítulo e imagem centralizada.

- **Rodapé**:
  - Informações de contato.
  - Links para redes sociais.

---

## ✅ Tecnologias Utilizadas
- **HTML5** e **CSS3**
- **Font Awesome** para ícones
- Design responsivo com **media queries**
- Animações CSS para transições suaves

---

## ✅ Como Executar Localmente
1. Baixe ou clone este repositório:
   ```bash
git clone https://github.com/seuusuario/seurepositorio.git
```
2. Extraia os arquivos (se baixou o ZIP).
3. Abra o arquivo `index.html` no navegador.

---

## ✅ Publicação no GitHub Pages
1. Suba os arquivos para o seu repositório.
2. Vá em **Configurações → Pages → Branch principal → /root**.
3. O site estará disponível no link gerado pelo GitHub Pages.
