# Site Débora Kulczar Advocacia

## Estrutura
- Header com imagem à esquerda e textos à direita.
- Seção Áreas de Atuação com blocos responsivos.
- Bloco Advogada Responsável com foto.

## Como usar
1. Adicione as imagens: `escritorio-moderno.jpg` e `perfil.jpg` na pasta do projeto.
2. Substitua `logo.png` se necessário.
3. Abra `index.html` no navegador ou publique no GitHub Pages.

## Personalização
- Cores: ajuste no `style.css`.
- Textos: edite no `index.html`.
